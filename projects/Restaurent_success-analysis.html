Project Overview

This project analyzes Yelp restaurant businesses using data from an SQLite database. It explores key business metrics, including review counts, star ratings, and geographical trends, leveraging Python for data processing and visualization.

The analysis investigates the distribution of restaurant reviews, identifies high-performing businesses, and maps restaurant locations using Folium. SQL queries retrieve structured insights, while data visualizations highlight patterns in customer feedback and engagement.

Data preprocessing includes filtering restaurants from the business dataset and handling missing values for accuracy. Review count distributions and descriptive statistics provide a comprehensive overview of business performance.

Key insights suggest that high-rated restaurants tend to receive more reviews, with location playing a significant role in customer engagement. Further segmentation by restaurant type and location density is recommended for deeper insights.
